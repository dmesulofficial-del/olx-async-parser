OLX Real Estate Async Parser
Опис
Асинхронний скрейпер для збору даних про нерухомість (квартири) з OLX.ua. Скрипт збирає 50+ оголошень, заходить у кожне для отримання детальних характеристик та експортує дані в Google Sheets.

Технології
Python 3.12

Playwright (Async API) — для імітації дій користувача та обходу динамічної верстки.

gspread / Google Sheets API — для запису даних.

Asyncio — для паралельної обробки сторінок (використано Semaphore для стабільності).

Як ми спарсили дані
Головна сторінка: Збір посилань, цін та назв міст (обробка рядка для отримання лише назви населеного пункту).

Сторінка оголошення: Використано точкові селектори для контейнера ad-parameters-container. Ми парсимо всі параграфи <p>, створюємо словник характеристик і витягуємо "Поверх", "Поверховість" та "Площу".

Оптимізація: Впроваджено випадкові паузи та обмеження в 1 асинхронний потік, щоб уникнути блокувань з боку анти-бот системи OLX.
